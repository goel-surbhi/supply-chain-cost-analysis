# Supply Chain Cost Analysis

## Overview
This project analyzes 100,000+ order and courier records to identify cost inefficiencies in supply chain operations.

## Tools & Technologies
- Python (Pandas, NumPy)
- Excel
- Power BI

## Key Insights
- Identified 15% overcharges and 10% undercharges
- Highlighted cost leakage areas
- Built KPI dashboards for decision-making

## How to Run
1. Open the Jupyter Notebook
2. Install required libraries
3. Run cells sequentially
